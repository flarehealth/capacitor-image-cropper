# iOS Plugin

## System Dependencies

 * macOS
 * Xcode
 * Ruby
    * Dependencies are managed via Cocoapods, which is installed via Ruby
 * Bundler

## Setup

1. Install Cocoapods

    ```bash
    bundle install
    ```

## Development

Make sure to open the `xcworkspace` file instead of the `xcodeproj` file for making changes to the plugin.
